# Astro

## School Management Dashboard

### Built with:
>  React Admin,

>  JSON-Server for our REST API

## Usage

```
# Install dependencies
yarn

# Install client dependencies
cd client
yarn
cd ..

# Run client and server (Visit http://localhost:3000)
yarn dev

# Run server only (:5000)
yarn server

# Run client only (:3000)
yarn client
```
